Querying Aspirin database

You can filter the results by:

  1. experiment (3H-Acetylation or Endogenous Acetylation)
  2. peptide expectancy threshold (none, 0.05, 0.5)
  3. maximum peptide length (if no value is entered, the filter has no effect)
  4. enzyme T or TG (if none value is entered, the filter has no effect)


About the peptide list

The peptide list is sorted by ascending rank product. The most significant peptide appears first in the list.

  1. Rank Product: score for each peptide to sort them by significance (the lowest the better).
    The + replicates are being sorted by descending the pep_score, while the - reps are being sorted by ascending the pep_score. This is done to get lowest (worst) rank for the peptides located in - replicates (penalty-like scoring). Each peptide found in a replicate could be found more than once, so the rank of the hit with the highest pep_score was chosen. If a peptide is not found in a replicate means that it was highly impossible so it gets by default the lowest rank in the replicate.
  2. Penalty: additional penalty applied on the rank product.
    Multiplied by 10 the ranks in the - replicates. This penalty is not applied if a peptide is not found in a replicate, it just gets by default the lowest rank in the replicate.


Getting more information about the peptide spectrum matches

Click on each peptide sequence, to view all the matches found for this peptide, taking into account the preselected filters. The psms are sorted by peptide score, in descending order.

  1. Query: a mascot search for a single MS/MS spectra against the protein database
  2. Protein accession: identifier to search through database entries
  3. Peptide score: a peptide spectrum match (PSM) is assigned a score based on the number and type of ions that are present in the spectrum
  4. Peptide expectancy:
  5. Replicate: the raw data file
  6. Modification: locations of all modified residues and actual modified residues. K(2) means that there is a modification in the amino acid K that is in the second position through the peptide.
  7. Title: spectrum identification
  8. Number of spectrums in which each ion is assigned as the most intense.
  9. Spectrum: mass spectrum labelled with fragment ions. The figure for each psm that represents the intensities over the amino acid masses through the peptide. Red peaks are the significant ones. With black peaks is the full spectrum.
  10. Matched fragment ions table: the significant transitions. The masses are shown in a combined format of: theoretical mass (assigned experimental mass : assigned intensity). The 3 most intense peaks are bolded.
  11. There are multiple psms for each peptide.


Getting more information about the protein

Click on a protein accession number in a PSM, to view details like species, genename, protein description and sequence, as well as the conservation of the modification through various species.

  1. Sequence: The peptide and its modifications highlighted through the protein sequence. The peptide is highlighted with red and the modifications with blue.
  2. Multiple alignment: The acetyl amino acid modification highlighted to view if it is conserved through different species.

    The species aligned are:
    1. hg19 (human)

    4. mm9 (mouse)

    6. bosTau4 (cow)
