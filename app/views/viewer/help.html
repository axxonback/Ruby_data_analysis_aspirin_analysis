Help

Some help information on how to use this site, and what the results page mean.


Querying Aspirin database


  1. There are two options to filter the results. One is the name of the experiment (3H-Acetylation or Endogenous Acetylation) and second the cutoff (none, 0.05, 0.5) for the peptide expectancy value.
  2. You have to select both filters.
  3. By default if you choose no filtering, the results viewed are from the 3H-Acetylation experiment, and include the full peptides list, with no cutoff applied.


About the peptide list


  1. Rank Product: score each peptide to sort them by significance (the lowest the better)
  2. Penalty: additional small penalty in the rank product


Getting more information


If you click on each peptide, you will see all the psms that apply for this peptide, taking into account the preselected filters.

  1. Query:
  2. Protein accession: identifier to search through database entries
  3. Peptide score:
  4. Peptide expectancy:
  5. Replicate: the raw data file
  6. Modification: locations of all modified residues and actual modified residues
  7. Title: spectrum identification
  8. Spectra: the figure for each psm that represents the intensities over the amino acid masses through the peptide. Red peaks are the significant ones. With blue peaks is the full spectrum.
  9. Assigned y-ions table: the table that contains only the significant transitions, keeping track of their index and the theis m/z and intensity values.
  10. There are multiple psms for each peptide.
