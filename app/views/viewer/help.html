Help

Some help information on how to use this website, and what the results pages mean.


Querying Aspirin database

There are two options to filter the results. One is the name of the experiment (3H-Acetylation or Endogenous Acetylation) and second the cutoff (none, 0.05, 0.5) for the peptide expectancy value.


About the peptide list

  1. Rank Product: score each peptide to sort them by significance (the lowest the better)
  2. Penalty: additional small penalty in the rank product


Getting more information about the peptide spectrum matches

Click on each peptide, to view all the psms that apply for this peptide, taking into account the preselected filters.

  1. Query: a search for a single MS/MS spectra against the protein database
  2. Protein accession: identifier to search through database entries
  3. Peptide score: a peptide spectrum match (PSM) is assigned a score based on the number and type of ions that are present in the spectrum
  4. Peptide expectancy:
  5. Replicate: the raw data file
  6. Modification: locations of all modified residues and actual modified residues. K(2) means that there is a modification in the amino acid K that is in the second position through the peptide.
  7. Title: spectrum identification
  8. Spectra: the figure for each psm that represents the intensities over the amino acid masses through the peptide. Red peaks are the significant ones. With blue peaks is the full spectrum.
  9. Assigned y-ions table: the table that contains only the significant transitions, keeping track of their index and the theis m/z and intensity values.
  10. There are multiple psms for each peptide.


Getting more information about the protein

Click on a protein accession number in a PSM, to view details like species, genename, protein description and sequence, as well as the conservation of the modification through various species.

  1. Sequence: The peptide and its modifications highlighted through the protein sequence. Red highlight for the peptide, blue for the modifications.
  2. Multiple alignment: The acetyl amino acid modification highlighted to view if it is conserved through different species.

    The species aligned are:
    - hg19 (human)
    - rn4 (rat)
    - oryCun2 (rabbit)
    - mm9 (mouse)
    - gorGor1 (gorilla)
    - bosTau4 (cow)
    - danRer6 (zebrafish)
    - galGal3 (chicken)
